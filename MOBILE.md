# Мобильная версия BizAuto

## Реализованные возможности

### 🎨 Дизайн в стиле Claude.ai
- Минималистичный и чистый интерфейс
- Светлая цветовая схема с акцентом на зелёный
- Много воздуха и пространства между элементами
- Плавные скругленные углы

### 📱 Мобильная адаптация
- **Адаптивное меню**: Гамбургер-меню на мобильных устройствах (< 1024px)
- **Боковая панель**: Выдвигается слева с плавной анимацией
- **Оверлей**: Затемнение фона при открытом меню
- **Закрытие**: По клику вне меню, на крестик или клавишу Escape

### 🎯 Оптимизации для мобильных
- Адаптивная типографика (14px на мобильных, 16px на десктопе)
- Увеличенные области нажатия (минимум 44x44px)
- Предотвращение зума на iOS при фокусе на input (font-size: 16px)
- Отключение подсветки при нажатии (-webkit-tap-highlight-color)
- Поддержка safe-area для устройств с вырезами

### ✨ Анимации и переходы
- Плавное выдвижение меню (300ms cubic-bezier)
- Fade-in для оверлея
- Smooth scroll для всей страницы
- Оптимизированные hover-эффекты только для десктопа

### 📐 Брейкпоинты
- **Mobile**: < 768px
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px
- **Large Desktop**: > 1920px

### 🎨 Стилевые улучшения
- Уменьшенные отступы на мобильных (padding: 1rem вместо 1.5rem)
- Компактные карточки и кнопки
- Адаптивная высота хедера (60px на мобильных)
- Оптимизированные модальные окна (85vh на мобильных)

### ♿ Доступность
- Правильные ARIA-метки для кнопок
- Focus-visible стили для клавиатурной навигации
- Поддержка prefers-reduced-motion
- Поддержка prefers-contrast: high

## Как использовать

1. Откройте сайт на мобильном устройстве или уменьшите окно браузера
2. Нажмите на иконку меню (☰) в левом верхнем углу
3. Выберите нужный раздел
4. Меню автоматически закроется после выбора

## Тестирование

### Chrome DevTools
1. Откройте DevTools (F12)
2. Нажмите Toggle Device Toolbar (Ctrl+Shift+M)
3. Выберите устройство или установите кастомный размер

- Android (Chrome)
- iPad (Safari)
