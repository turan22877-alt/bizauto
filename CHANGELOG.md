# Changelog - Обновления системы

## 2026-04-11 - Крупное обновление функционала

### ✅ Добавлено

#### 1. Система видов услуг
- **Новый раздел "Услуги"** в меню (Управление → Услуги)
- Возможность создавать, редактировать и удалять услуги
- Каждая услуга содержит:
  - Название
  - Категория
  - Длительность (мин)
  - Стоимость
  - Описание (опционально)
- Группировка услуг по категориям
- Интеграция с журналом записей - выбор услуги из списка с автозаполнением цены и длительности

#### 2. Технологические карты
- **Новый раздел "Техкарты"** в меню (Управление → Техкарты)
- Привязка расходных материалов к услугам
- Автоматическое списание материалов со склада при подтверждении записи
- Расчет себестоимости услуги по материалам
- Возможность добавлять несколько материалов на одну услугу

#### 3. Улучшенный журнал записей
- **Множественные записи на одно время**: теперь несколько клиентов могут быть записаны на одно время (карточки отображаются рядом)
- **Уведомления по времени**: автоматические уведомления о предстоящих записях (за 30 минут)
- Интеграция с системой услуг - выбор из справочника
- Автоматическое списание материалов по техкартам

#### 4. Расширенная аналитика
- **Продажи по сотрудникам**: таблица с выручкой и количеством записей каждого сотрудника
- **История клиентов**: детальная история услуг и покупок каждого клиента
- **Остатки на складе**: таблица с текущими остатками товаров и их стоимостью
- **Экспорт данных**: новые CSV-отчеты
  - Продажи по сотрудникам
  - История клиентов
- Все данные с разбивкой по месяцам (последние 12 месяцев)

### 📊 Новые возможности аналитики

1. **Продажи по сотрудникам**
   - Количество записей
   - Общая выручка
   - Сортировка по убыванию выручки

2. **История клиентов**
   - Все оказанные услуги
   - Даты и стоимость
   - Общая сумма потраченных средств
   - Количество визитов

3. **Учет материалов**
   - Остатки на складе
   - Стоимость остатков
   - Автоматическое списание по техкартам

### 🎯 Как использовать

#### Настройка услуг:
1. Перейдите в "Управление → Услуги"
2. Нажмите "Добавить услугу"
3. Заполните данные: название, категория, длительность, цена
4. Сохраните

#### Создание техкарты:
1. Перейдите в "Управление → Техкарты"
2. Нажмите "Создать техкарту"
3. Выберите услугу
4. Добавьте материалы и их количество
5. Сохраните

#### Работа с записями:
1. В журнале записей выберите услугу из списка (автозаполнение цены/времени)
2. При подтверждении записи материалы автоматически спишутся со склада
3. Уведомления о предстоящих записях появятся за 30 минут

#### Просмотр аналитики:
1. Перейдите в "Аналитика → Аналитика"
2. Просмотрите графики выручки и записей по месяцам
3. Изучите продажи по сотрудникам
4. Проверьте историю клиентов
5. Экспортируйте нужные отчеты в CSV

### 🔄 Миграция данных

Все существующие данные сохранены и работают как прежде. Новые поля добавлены опционально:
- Записи без привязки к услугам продолжают работать
- Техкарты применяются только к новым подтвержденным записям
- Старые резервные копии совместимы (новые поля игнорируются при отсутствии)

### 📝 Примечания

- Уведомления проверяются каждую минуту
- Материалы списываются только при статусе "Подтверждено"
- Множественные записи отображаются рядом друг с другом
- Все новые данные включены в систему резервного копирования
